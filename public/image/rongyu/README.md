# 图片资源管理规范

本目录用于存放项目所需的各类荣誉资质图片。为确保资源的一致性和可维护性，请严格遵守以下管理规范。

## 1. 目录结构

所有图片应存放在 `public/image/rongyu/` 下的相应子目录中：

*   **`iso/`**: 存放 ISO 认证相关证书图片（如：质量管理体系、环境管理体系等）
*   **`shangbiao/`**: 存放商标著作权证书图片
*   **`ruanzhu/`**: 存放软件著作权证书图片
*   **`chengxin/`**: 存放 AAA 信用等级证书图片
*   **`yuming/`**: 存放域名注册证书图片
*   **`rongyu/`**: 存放其他企业荣誉奖项相关图片

## 2. 文件命名规范

采用 **`证书类型_编号_日期`** 的格式，使用下划线分隔。

*   **证书类型**: 使用英文简写或拼音首字母（如 `ISO`, `SB`, `RZ`, `AAA`）。
*   **编号**: 证书编号或序列号（如 `001`, `202301`）。
*   **日期**: 颁发日期，格式为 `YYYYMMDD`。

**示例**:
*   `ISO_9001_20230615.jpg`
*   `RZ_Software_20231201.png`
*   `AAA_Credit_20230101.jpg`

## 3. 文件格式要求

*   **格式**: 统一保存为 **JPEG (.jpg)** 或 **PNG (.png)** 格式。
*   **分辨率**: 不低于 **300dpi**，确保打印和高清屏显示效果。
*   **文件大小**: 单张图片严格控制在 **5MB** 以内。建议在保证清晰度的前提下进行适当压缩，以优化网页加载速度。

## 4. 使用说明

在代码中引用图片时，请使用以 `/` 开头的绝对路径（相对于 `public` 目录）：

```tsx
// 示例
<img src="/image/rongyu/iso/ISO_9001_20230615.jpg" alt="ISO证书" />
```

请定期清理不再使用的旧图片，保持目录整洁。
